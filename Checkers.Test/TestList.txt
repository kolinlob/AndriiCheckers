== GAME
* can create game
game checks, if a checker can move to that field
checker can beat the opponent's checker
switches player after each move
checks for winner

== BOARD
* can create board
* can display board
board can draw checker

== PLAYER
player can choose their checker
player can set coordinates for a checker to move
player can make a move
player can surrender =)

== CHECKER
* can create a checker
checker can move
checker can become queen
checker changes its appearance, when becomes queen
checker moves according to the coordinates set
